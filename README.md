# Testes Automatizados com Cypress - Rede Integra e Portal Integra

# Tecnologias usadas
- [Cypress](https://www.cypress.io/) - Framework de testes end-to-end
- Node.js
- JavaScript

# Pré- requisitos
  - Node.js 
  - npm
    
# Para abrir o Cypress em modo interativo:
npx cypress open

# Para rodar os testes no terminal:
npx cypress run
